# Standard test case header

Copy into new cases. Bootstrap cases include the review banner until approved.

```markdown
> **Bootstrap — review required.** Expected results must be re-derived from
> `docs/SPEC.md` and companion guide PDFs only. See `qa-uat/PROVENANCE.md`.

| Field | Value |
|-------|-------|
| **Review status** | `pending` / `approved` |
```

When approved, replace banner with:

```markdown
| **Review status** | `approved` |
| **Reviewed by** | |
| **Review date** | |
| **Authority** | SPEC + companion guide (cite sections in Expected results) |
```
